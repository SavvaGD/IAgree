📄 IAgree

Сервис для подписания согласий с переменными

IAgree позволяет создавать шаблоны согласий с переменными {...} и подписью <...>, отправлять их клиентам на подпись и получать готовые документы.

  1. Админ загружает шаблон .txt с переменными {firstname} и подписью <signature>
  2. Сервис генерирует .iagreensign — бинарный файл для клиента
  3. Клиент загружает .iagreensign, заполняет поля и подписывает
  4. Сервис выдаёт .json с данными и .iagreesign — заполненный документ
  5. Админ загружает .iagreesign и скачивает готовый .txt
